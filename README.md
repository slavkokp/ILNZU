# ILNZU
Web chatting application written on .NET CORE 5 using Entity Framework 5, signalR, xunit, serilog and PostgreSQL as DBMS.

Initial setup
Create database ILNZU_database in postgres (edit connection string here in[a link](https://github.com/slavkokp/ILNZU/blob/master/ILNZU/ILNZU/appsettings.json)
Write Update-Database in package manager console to update database structure.
Run using ILNZU configuration.
